Technologies used

- HTML5

- CSS3 (Flexbox, Grid)

- Bootstrap5 

- Visual Studio Code

- Figma

Features

- Responsive design for mobile devices.


How to use

Prerequisites

- Git
- Web browser


Installation

1. Clone the repository:

  git clone https://github.com/rafaelmprogrammer/bim_the_game_website.git

2. Access the directory:

  cd my-site

3. Open the index.html file in your browser
